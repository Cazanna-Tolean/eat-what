## Getting Started

### Prerequisites

- Node.js version
  ^12.14.1 || ^14.15.0

- TypeScript version
  ~4.2.4

- Angular Cli version
  ~12.0.5

### Installation

1. Clone the repo

```sh
git clone https://github.com/Cazanna-Tolean/eat-what.git
```

2. Install NPM dependency

```sh
npm install
```

### How to Use?

1. Revise the Mock data

file path

```
src\app\data\mockData.ts
```

2. run the project

```sh
ng serve
```
